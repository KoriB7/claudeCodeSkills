# Claude Code Skills

My custom skills for Claude Code CLI.

## Skills

### axon-review
Reviews SkySpark Axon code for best practices, efficient Haystack querying, proper function design, performance optimization, and common pitfalls.

**Use when:** Reviewing Axon scripts, functions, or queries in SkySpark/Project Haystack context.

### github-sync
Sync Claude Code skills to GitHub repository.

**Use when:** Saving, committing, pushing, or backing up skills to git/GitHub.

## Installation

To use these skills, copy them to your Claude Code skills directory:

**Personal skills** (available across all projects):
```bash
~/.claude/skills/
```

**Project skills** (specific to this project):
```bash
.claude/skills/
```

## Usage

These skills are automatically detected and used by Claude Code when relevant to the task at hand.

## About

These skills help automate and standardize common tasks when working with SkySpark and development workflows.
